# OPM Fingerstyle — Submission and Admin Workflow

## Submission flow
1. creator opens `/submit`
2. creator fills out form
3. submission is stored as `pending_review`
4. admin reviews submission
5. admin either approves, edits, or rejects
6. approved submission becomes public

## Submission fields
Required:
- display name or artist name
- short bio
- profile photo
- YouTube channel URL or at least one YouTube video URL

Optional:
- location
- tab links
- social links
- contact info
- additional videos

## Validation rules
- reject empty or low-effort submissions
- reject clearly irrelevant content
- reject malformed URLs
- reject duplicate or spam submissions

## Admin capabilities
Admin must be able to:
- view pending submissions
- edit guitarist fields
- approve submission
- reject submission
- remove broken video/tab/social links
- mark product as sponsored
- manage affiliate URLs
- create and publish articles

## Moderation rules
Publish only when:
- profile is relevant to OPM/fingerstyle
- content is not spam
- minimum required fields exist
- page has enough content to be useful
